Descendant selectors

If you right-click here and then click inspect, you might be able to find the CSS settings I've used for this webpage. You can then find something I want to talk about.

Pay attention to the part I've circled in yellow. Inside is an example of an ancestor-descendant selector, which means that all elements that fit the selected criteria will be affected by the CSS styling.

If I were to put a > symbol in the selector between div and p, only the first paragraph in each div element will be affected. So this paragraph down here would use default styling instead of the style I've set, and that might be a little hard to read against this background I made (I tried my best to make it legible and it still wasn't enough, ok?)
